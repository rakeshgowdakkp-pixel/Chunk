# Chunks
**Text Chunking for RAG Systems**

This project demonstrates how to split large text into smaller pieces called chunks using Python.

Text chunking is an important step in Retrieval-Augmented Generation (RAG) systems because large language models cannot process very large documents at once. Instead, documents are divided into smaller chunks


**Learning Outcome**

Through this project I learned:

    How to split large text into smaller chunks

    Why chunking is necessary for RAG pipelines

    Preparing text for embedding models
